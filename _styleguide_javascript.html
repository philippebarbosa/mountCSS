Tutorial : How to make dropdown link clickable :

  • Add disabled and has_submenu_link class to the element. has_submenu_link purpose is to avoid bugs with mobile.
  • Clickable + submenu content visible :
    • Add disabled class to the element.
    • Uncomment this css at the end of _multimenu.scss : 
          
        .header .navbar {
            @media (min-width : $desktop ) {
                .dropdown-menu {
                    display: block;
                    opacity: 0;
                    visibility: hidden;
                    transition: .1s all ease-in;
                }
                .navbar-nav > li:hover > .dropdown-menu {
                    opacity: 1;
                    visibility: visible;
                }
            }
            .dropdown-menu li {
                transition: .2s all ease-in;
            }
        }
    • Uncomment this js in index.js. It allows submenus to be opened on mobile device. 
          
        $('.has_submenu_link').removeClass('disabled');
